師大特選

發表於2023-11-18|更新於2024-02-09
|閱讀量:|評論數:

❀ 前言 ❀

「又得重返獨自一人泡在圖書館的日子了」一邊騎腳踏車一邊想。
凌冽的寒風劃過我的臉龐，我是這麼的清醒，但回憶卻如此的虛幻，昨天的一切，就像一場夢。
是的，我成功進入師大資工的特殊選才二階面試了。

❀ 正題 ❀

<報到>

一開始，真的滿緊張的，我去散了步，伸展一下自己。
不一會兒，熱情的系主任方瓊瑤教授就出現了，聊天的過程中我更了解師大，也稍微比較放鬆了
後來也有遇到李忠謀教授，其實之前就看過他了，但我現在才知道他叫什麼:P
師大的老師都很熱情友善，害我更想進師大了> <

<面試>

聊著聊著，不知不覺就輪到我了，在進教室前還能跟師大的前輩們聊個天。
話說這個時候我學長好像剛好去休息師看我，結果沒遇到QQ
總之，自介加問答挑戰開始！

[第一間]

三分鐘自介，順順的=v=
教授ㄉ提問：

  • 你在你的遊戲專案中最難的部分是什麼，你如何解決

    我覺得最困難的部分是使用架構，之前我寫得小專案都是所有程式碼寫在同一個腳本中，但 unity 卻不然，因此我研究了MVP架構。
    M 是 Model 處理邏輯，V 是 View 處理畫面 UI 顯示，P 是 Presenter 當作兩者之間的橋樑。

  • 你資訊能力競賽解出幾題，印象最深刻的是什麼

    (這邊講得有點爛，因為忘記題目了。)
    我解出的是卡片問題
    可以想成 an, an+1 相比，假設an > an+1，
    那此刻，最少的罰分就是an，找不到比他更小的，
    因為若an+1先跟他右邊的比，結果am > an > an+1而an+1被刪除後替換成了am，此時罰分增加。
    所以左右相比就能得到最小罰分。

  • 所以你用了什麼資料結構阿？

    這題我認為只需要先思考，用邏輯就解出來了，不太需要演算法。

  • 你會什麼資料結構

    vector、stack、queue、deque…

  • 你會樹嗎（一種結構）

    ㄜ不太熟

[時間到]

[第二間]

桌上有九張牌，寫了一些關鍵字

  • 跟我用一分鐘簡單介紹一下你自己

    我是政大附中數理資優班的許育綸，
    我的特質是1用程式解決問題的能力2遊戲專案的經驗3熱忠於傳播知識，
    因為我是資訊社的社長兼教學，所以我教過滿多課的

  • 桌上有九張牌，你挑你喜歡的翻過來回答問題。
  • [表達]：用兩分鐘跟高中同學介紹你熟悉的知識點

    我想介紹 python 裡的容器，用交通工具比喻
    set 像重機，只能在獨一無二的女朋友，所以沒有重複元素
    tuple 像 uber 訂了就不能換座位，所以不能改變元素
    list 像公車，可以任意新增元素
    dictionary 像火車要會號入座，一個 key 對一個 value

  • 滿有創意的，你再翻一張
  • [未來]：你未來想做什麼，要如何實踐

    我未來想當遊戲工程師，因為我發現遊戲設計不只要會蟹程式，也要會理解玩家心理，與編寫好的結構
    在大學我會想製作更多遊戲作品集，並參加 Citcon 或 GameJam 跟更多人交流切磋

  • 你遊戲都是獨自開發的嗎

    對阿哈哈，但這也是為什麼我想跟學長一起創辦政附資訊社群，能整合政大附中的資訊人才，並跟更多人交流切搓。
    我也有在裡面找到一個第九屆的學長，他目前在遊戲介工作，跟他聊我才了解程式的架構比寫成是重要。

  • 你參加過 GameJam 嗎

    我只看過相關影片耶
    ㄟ對，我能問你問題ㄇ

  • 當然可以，問吧

    學校有哪方面跟遊戲開發有關的嗎，如果我想開發遊戲，我應該修什麼

  • 我們學校的音樂很厲害，之後可以跨系跟他們合作，另外我們也有學長正在雷亞工作喔。

    喔喔，剛剛跟系主任有聊到說師大主打擴展自己的多面向發展。

[時間到]

  • 那時間關係就先聊到這裡囉

❀ 後話 ❀

第一間面的應該是林順喜教授、賀耀華教授(這我沒認出來)
第二間面的應該是蔣宗哲教授(這我沒認出來)
但，ㄏㄏ我也不敢叫名字裝熟。

回歸正題，一切的契機，是陳子騏的一句話，讓我有了報名特選的念頭，
然而我記得那隔天師大就開始報名了。
幸好中間遇到連假，賣了肝，請到了也是特選的學長與編輯老爸幫我檢查備審資料，
修了修改了改，糊里糊塗的壓著線，把郵件寄出去了。

在那之後我就冷凍特選了一陣子，畢竟還是得顧學測，直到模考後，我做了第一版的 ppt
並在校慶當天，進行了第一次的口試預演。
真的能過一階嗎？迷茫的我沒順過稿，沒掌握重點，想當然爾被臭罵了一頓，畢竟我找了學長與資訊組長來幫我聽
但也有如當頭棒喝，吵醒了內心的資訊魂，隔天，我生出了第二版簡報…
就這樣不停的重複喚醒自己的靈魂，簡報、講稿、問題集也陸續的更新，僅僅過了10天，我天天都有昇華的感覺。
超級感謝修佑學長跟芝敏組長，願意花這麼多時間跟我聊天和給予建議，並言簡意賅的點出我的缺點。
謝謝愷恩導師教我回答問題的技巧，面試真的用到啦。
謝謝我的家人，聽完我的報告給我獨特而有深度的問題。
謝謝幫我聽自我介紹的鈞哥和溫姐，也謝謝你們的回饋與鼓勵。
謝謝吳玉老師提醒我不要緊張，謝謝孝仁老大跟陳子騏給我信心。
也謝謝班上每個同學給我的支持（？）與建議（不是水果禮盒）。
最最最最最感謝的，還是我自己，謝謝你願意投入資訊領域，謝謝你一路走來願意付出這麼多，謝謝你不到最後一里路都不放棄。
真的太多太多人要感謝了，如果不是這麼多的巧合，也不會有如今的我。
總之，我覺得自己已經盡力了，不論結果是什麼，就是緣份了。
下集：１２／７日敬請期待！
